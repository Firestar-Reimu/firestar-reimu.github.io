申请 GitHub Pages

新建一个 repository，命名为 (user_name).github.io，隐私设置为 Public，再建一个 GitHub 仓库用来存放博客的源代码（可选）

申请域名

我选择的是阿里云，在万网输入自己想要的域名并购买。一般来讲 .top 最便宜，.xyz 次之。 假设域名为 example.top 根据网站提示做好域名实名制认证等流程。在域名控制台中选择：域名 >> 操作 >> 解析

解析中添加一条记录：

再去名为 (user_name).github.io 的 repository，新建一个文件，文件名为 CNAME，填入 blog.example.top ，通过 GitHub 的网络检测后就可以使用。

此时可以分别在浏览器中输入 (user_name).github.io 和 blog.example.top 试一试，如果能显示README.md 中的内容则设置成功。
